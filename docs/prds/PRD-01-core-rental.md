# PRD-01: Core Rental Platform (P0 — MVP)

## User Personas
**Renter (Thabo):** Weekend trips, backup vehicle. Pain: deposits R5k-R15k, 30min paperwork.
**Owner (Nosipho):** Car parked 20 days/month. Pain: security concerns, manual mgmt.

## Key Features
- **Search:** Location, date, price, make/model, transmission, fuel, seats
- **Quote:** Instant pricing with all fees (rate + insurance + WDR Shield)
- **Booking:** Trust-based deposit waiver, card/EFT payment, digital check-in/out
- **Check-in:** Selfie, odometer photo, fuel level, damage scan
- **Check-out:** Odometer, fuel, damage photos, flag issues
- **Payments:** Card tokenization (Yoco/Peach), auto-payout to owner in 48h
- **Reviews:** Both sides rate (1-5 stars)

## Success Metrics
- 50 bookings/day (MVP), Fleet utilization >30%, NPS >40
- Trust Alpha: 30% defect reduction vs traditional rental
- Deposit waiver adoption: 40% of eligible bookings
